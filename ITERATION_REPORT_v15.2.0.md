  - 一键复制完整代码

  - 控制台错误捕获和展示

## 四、性能优化

### 优化内容

1. **防抖自动运行**
   - WebIDE 代码修改后 500ms 防抖
   - 避免频繁的 iframe 重渲染
   - 提升编辑时的流畅度

2. **React 性能优化**
   - 使用 `memo` 包裹组件避免不必要重渲染
   - 使用 `useCallback` 稳定函数引用
   - 合理使用 `useRef` 避免状态更新

3. **控制台输出限制**
   - 控制台输出最多保留 50 条
   - 避免内存溢出和渲染卡顿
