# WebLinuxOS

一个完全运行在浏览器中的 Linux 桌面环境，基于 React + TypeScript 构建，提供完整的桌面体验和丰富的实用应用程序。这不是一个简单的模拟器，而是具有实际应用价值的 Web 操作系统。

## 功能特性

### 桌面环境
- **窗口管理系统** - 支持拖拽、缩放、最小化、最大化、关闭，窗口层级自动管理
- **任务栏** - 显示时间、系统托盘、运行中的应用，支持快速切换
- **开始菜单** - 分类应用列表、快速搜索、最近使用记录
- **桌面图标** - 双击打开应用，支持自定义排列
- **多主题支持** - 深色/浅色主题切换，多种壁纸选择
- **虚拟文件系统** - 完整的文件树结构，支持创建、删除、编辑操作

### 内置应用（60+）

#### 系统工具
- **文件管理器** - 完整的文件浏览和管理，支持树形目录、右键菜单
- **终端** - 功能完整的 Linux 命令行模拟，支持 20+ 常用命令
- **系统设置** - 主题、壁纸、显示设置，个性化配置
- **系统监视器** - 实时 CPU、内存、网络监控图表
- **任务管理器** - 进程管理，窗口状态控制
- **进程监视器** - 详细进程信息查看
- **网络监视器** - 网络连接状态监控
- **磁盘使用分析器** - 磁盘空间可视化
- **用户管理** - 用户账户配置

#### 实用工具
- **天气** - 实时天气查询，支持全球城市搜索（Open-Meteo API）

## 技术栈

- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite 6
- **状态管理**: Zustand
- **样式**: CSS Variables + 响应式设计
- **API集成**:
  - Open-Meteo (天气数据)
  - Frankfurter (汇率数据)
  - ipapi.co (IP地理位置)
  - Hacker News API (技术新闻)

## 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## API 数据来源

本项目集成了多个公开API提供真实数据：

| 功能 | API | 说明 |
|------|-----|------|
| 天气 | Open-Meteo | 免费、无需认证的天气API |
| 汇率 | Frankfurter | 欧洲央行汇率数据 |
| IP查询 | ipapi.co | IP地理位置信息 |
| 新闻 | Hacker News | Y Combinator技术新闻 |

## 项目亮点

1. **真实API集成** - 天气、汇率、IP查询、新闻等功能均使用真实公开API，提供实际数据
2. **完整窗口管理** - 支持窗口拖拽、缩放、层级管理、最小化/最大化，体验接近真实操作系统
3. **功能丰富的终端** - 支持 20+ Linux 常用命令，包括文件操作、系统信息查询等
4. **虚拟文件系统** - 完整的文件树结构，支持创建文件夹、新建文件、删除、重命名等操作
5. **开发者友好** - 提供JSON格式化、Markdown编辑器、开发者工具箱等实用工具
6. **生产力工具** - 番茄钟、待办事项、笔记等帮助提升工作效率
7. **轻量高效** - 基于 Vite 构建，首屏加载快速，运行流畅
