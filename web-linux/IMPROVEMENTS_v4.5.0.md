## 新增功能

### 1. 高级数据可视化应用 (DataVisualizer)
全新的数据可视化工具，提供以下功能：
- **四种图表类型**：柱状图、折线图、面积图、饼图
- **预设示例数据**：销售数据、用户增长等常用场景
- **自定义数据**：支持添加/编辑数据系列和数据点
- **图表导出**：支持导出为 SVG 和 PNG 格式
- **JSON数据管理**：可导入/导出数据为 JSON 格式
- **样式配置**：支持自定义颜色、标题、坐标轴标签等
- **网格和图例**：可配置是否显示网格线和图例
